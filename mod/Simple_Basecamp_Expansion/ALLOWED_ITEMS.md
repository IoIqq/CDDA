## 使用示例

### ✅ 正确示例
```jsonc
"components": [
  [ [ "2x4", 20 ] ],      // 使用白名单中的物品
  [ [ "nail", 40 ] ],     // 数量合理
  [ [ "log", 10 ] ]       // 可获取的材料
]
```

### ❌ 错误示例
```jsonc
"components": [
  [ [ "advanced_steel", 10 ] ],  // ❌ 不在白名单中
  [ [ "magic_wood", 5 ] ],       // ❌ 自创物品
  [ [ "2x4", 10000 ] ]           // ❌ 数量不合理
]
```
