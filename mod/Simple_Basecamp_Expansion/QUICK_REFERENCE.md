# Simple Basecamp Expansion - 快速参考指南

## 🚀 快速开始

### 第一步：建造图书馆
```
营地任务 → 建造 → 图书馆
材料：木材、钉子
时间：约2小时
```
**重要**：图书馆是所有研究的前置条件！

### 第二步：建造基础生产建筑
```
推荐顺序：
1. 简单铁匠铺（金属加工）
2. 简单木工坊（木工制作）
3. 简单温室（食物生产）
```

### 第三步：开始研究
```
收集材料 → 执行研究任务 → 获得研究报告 → 建造高级建筑
```

## 🔧 常用命令速查表

### 营地任务
```
按 * 键 → 营地任务菜单
选择建造 → 选择建筑
分配NPC → 等待完成
```

### 研究任务
```
前提：图书馆 + 基础建筑
准备材料 → 营地任务 → 研究
等待完成 → 获得研究报告
```

### 对话NPC
```
遇到编年史记录员
按 C 键对话
选择"给我讲讲故事吧"获得编年史
选择"我想查看我们的进度"查看目标
```

## ⚠️ 常见问题速查表

### Q: 研究任务找不到？
**A**: 检查是否建造了图书馆和对应的基础建筑

### Q: 高级建筑无法建造？
**A**: 确认是否拥有对应的研究报告

### Q: 高级配方无法使用？
**A**: 确认是否建造了对应的高级建筑

### Q: NPC在哪里？
**A**: 编年史记录员需要手动招募或通过特定方式生成

### Q: 如何调整难度？
**A**: 编辑 `config/mod_config.json` 中的 `active_preset`
