# 战利品处理中心使用指南

## 📦 功能概述

战利品处理中心是一个专门用于管理战斗任务后战利品的营地扩展建筑。它提供了自动化的物品集中和尸体处理功能，让您的营地管理更加高效。

---

## 🏗️ 建造要求

### 前置条件
- 营地等级：**4级或以上** (`faction_base_camp_4`)
- 建造位置：field 地形

### 所需材料
- 40 个 2x4 木板
- 20 个原木
- 80 个钉子
- 10 个金属板
- 30 个石头

### 所需工具
- 2级锤子
- 1级锯子
- 2级挖掘工具

### 建造时间
- **8小时**（480分钟）
- 技能要求：制造 4级
- 难度：4

---

## 🎯 建筑布局

战利品处理中心是一个 **10x10** 的建筑，包含以下区域：

```
布局示意图：
WWWWWWWWWW
WBB......W    B = 床位（2个）
W..PPP...W    P = 焚烧坑（3x3区域）
W..PPP.RRW    T = 工作台
W..PPP.RRW    R = 货架（多个）
WT.....RRW    L = 储物柜（多个）
WL.LLLL.RW    C = 木箱（多个）
WL.CCCC.RW    W = 木墙
WLLLLLLLRW    D = 门
WWWWDWWWWW
```

### 功能区域

1. **焚烧坑区域**（左上，3x3）
   - 专门用于堆放僵尸尸体
   - 可以在此处焚烧尸体
   - 地形类型：坑（t_pit）

2. **战利品整理区**（右侧和下方）
   - 多个货架：存放武器、工具
   - 多个储物柜：存放弹药、材料
   - 多个木箱：存放杂物
   - 工作台：用于拆解和整理物品

3. **生活区**（左上角）
   - 2个床位供NPC休息

## 🎮 使用方法

### 1. 建造战利品处理中心

1. 在营地菜单中选择"扩展营地"
2. 找到"战利品处理中心"选项
3. 选择建造位置（field地形）
4. 准备好材料和工具
5. 开始建造（需要8小时）

### 2. 执行战斗任务

建造完成后，所有战斗任务的描述都会更新：

- ✅ **清理周边 - 小规模**
- ✅ **清理周边 - 中等规模**
- ✅ **清理周边 - 大规模**
- ✅ **尸潮挑战**

任务描述会显示：
> "如果建造了战利品处理中心，物品将自动放置到指定区域。"

### 3. 战利品自动分类

战斗任务完成后，物品会通过 `blueprint_resources` 系统自动放置：

- **尸体** → 自动堆放到焚烧坑区域
- **武器和弹药** → 放置到货架上
- **工具和材料** → 放置到储物柜中
- **其他物品** → 放置到木箱中

### 4. 整理战利品（可选任务）

如果需要手动整理周围的战利品，可以使用：

**任务名称**：整理战利品  
**时间**：30分钟  
**要求**：建造了战利品处理中心  
**效果**：NPC会收集营地周围的物品和尸体，整理到处理中心

### 5. 焚烧尸体

当焚烧坑中堆积了大量尸体时，可以执行焚烧任务：

**任务名称**：焚烧尸体  
**时间**：2小时  
**要求**：
- 建造了战利品处理中心
- 点火工具（1级）
- 燃料（选择其一）：
  - 10个原木
  - 50个木棍
  - 30个枯枝

**效果**：焚烧坑中的尸体将被烧毁，清理空间

## 💡 使用技巧

### 最佳实践

1. **优先建造**
   - 建议在开始大规模战斗任务前建造
   - 可以避免战利品散落各处

2. **定期焚烧**
   - 尸体会占用空间并产生气味
   - 建议每次战斗后及时焚烧
   - 准备充足的燃料

3. **物品分类**
   - 定期检查货架和储物柜
   - 将重要物品转移到主仓库
   - 不需要的物品可以拆解或丢弃

4. **配合其他建筑**
   - 与**简单仓库**配合使用效果更佳
   - 与**武器维修站**配合，方便维修战利品武器
   - 与**弹药工坊**配合，可以回收弹药材料

### 空间管理

战利品处理中心提供的存储空间：
- 货架：6个（f_rack）
- 储物柜：8个（f_locker）
- 木箱：4个（f_crate_c）
- 工作台：1个（f_workbench）

总存储容量非常可观，足以应对大规模战斗的战利品。

## 🔧 解锁的功能

建造战利品处理中心后，会解锁以下 `blueprint_provides`：

1. **loot_processing**（战利品处理）
   - 允许执行"整理战利品"任务
   - 战斗任务奖励会自动放置到此处

2. **corpse_disposal**（尸体处理）
   - 允许执行"焚烧尸体"任务
   - 提供专门的尸体处理区域

3. **sorting**（分类整理）
   - 提供物品分类功能
   - 与其他仓库系统兼容

4. **bed**（床位）
   - 提供2个床位
   - 增加营地容纳能力

## ⚠️ 注意事项

1. **焚烧坑安全**
   - 焚烧时会产生烟雾和热量
   - 确保周围没有易燃物
   - 不要在焚烧坑附近存放重要物品

2. **存储容量**
   - 虽然存储空间很大，但仍需定期整理
   - 建议将贵重物品转移到主仓库
   - 定期清理不需要的物品

3. **NPC安全**
   - 焚烧任务需要NPC执行
   - 确保NPC有足够的生存技能
   - 准备好灭火工具以防万一

4. **与战斗任务的配合**
   - 战斗任务必须在建造处理中心后执行
   - 之前的战利品不会自动转移
   - 可以使用"整理战利品"任务手动收集

## 📝 常见问题

**Q: 战利品处理中心可以升级吗？**  
A: 目前版本暂不支持升级，但提供的空间已经足够使用。

**Q: 可以建造多个战利品处理中心吗？**  
A: 可以，但通常一个就足够了。多个处理中心会分散战利品。

**Q: 焚烧尸体会获得什么？**  
A: 焚烧尸体主要是为了清理空间，不会获得额外物品，但会产生少量灰烬。

**Q: 如果不建造战利品处理中心会怎样？**  
A: 战斗任务仍然可以执行，但战利品会直接进入营地库存，需要手动整理。
