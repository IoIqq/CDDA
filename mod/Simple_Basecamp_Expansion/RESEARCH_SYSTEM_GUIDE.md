# 研究系统使用指南

## 📚 概述

研究系统允许您通过消耗书籍和材料来获得研究报告，这些报告可用于建造高级建筑或解锁特殊配方。这是一个基于物品的科技进步系统，完全符合CDDA的营地机制。

---

## 🔬 系统原理

### 核心机制
```
收集书籍 → 执行研究任务 → 获得研究报告 → 建造高级建筑
```

### 研究报告
研究报告是特殊物品，代表您在某个领域的研究成果：
- 可以存储在营地库存中
- 用作高级建筑的建造材料
- 无法通过其他方式获得

## 🎮 使用流程

### 步骤1：准备前置建筑
1. 建造**图书馆**（所有研究都需要）
2. 建造相关领域的**基础建筑**
   - 例如：研究冶金需要铁匠铺

### 步骤2：收集研究材料
1. **书籍**：从废墟、图书馆、学校等地搜寻
2. **材料**：根据研究领域准备相应材料
3. **存储**：将材料放入营地库存

### 步骤3：执行研究任务
1. 打开营地任务菜单
2. 找到"研究：XXX"任务
3. 分配NPC执行任务
4. 等待研究完成

### 步骤4：获得研究报告
- 任务完成后，研究报告会自动添加到营地库存
- 检查库存确认获得报告

### 步骤5：应用研究成果
1. **建造高级建筑**：使用研究报告作为材料
2. **解锁特殊配方**：某些配方需要研究报告

## 💡 使用技巧

### 优先级建议

**新手玩家**：
1. 木工研究 → 提升基础建造能力
2. 农业研究 → 保证食物供应
3. 医学研究 → 提升生存能力

**中期玩家**：
4. 冶金研究 → 制作更好的工具和武器
5. 纺织研究 → 制作防护装备
6. 机械研究 → 提升生产效率

**后期玩家**：
7. 化学研究 → 制作高级药物和炸药
8. 电子研究 → 制作自动化设备

### 书籍获取

**常见地点**：
- 图书馆：各类教科书
- 学校：教科书和手册
- 书店：各类书籍
- 实验室：专业书籍
- 住宅：手册类书籍

**稀有书籍**：
- 大学：高级教科书
- 研究所：专业技术书籍
- 军事基地：军事技术手册

### 材料管理

**建议**：
1. 在营地建造**仓库**存储材料
2. 使用**标签系统**分类存储
3. 定期整理，保持库存清晰
4. 优先存储稀有材料

### NPC分配

**技能匹配**：
- 高制造技能NPC → 冶金、木工、机械研究
- 高化学技能NPC → 化学研究
- 高电子技能NPC → 电子研究
- 高急救技能NPC → 医学研究
- 高生存技能NPC → 农业研究
- 高裁缝技能NPC → 纺织研究

**效率提升**：
- 高技能NPC研究速度更快
- 高智力NPC成功率更高
- 多个NPC可以同时进行不同研究

## ⚠️ 注意事项

### 研究报告管理

1. **不可复制**
   - 研究报告无法通过制作获得
   - 只能通过研究任务获得
   - 使用后消失

2. **妥善保管**
   - 存储在安全的地方
   - 避免意外丢失
   - 建议建造前备份存档

3. **计划使用**
   - 确定发展方向后再使用
   - 优先建造最需要的建筑
   - 考虑长期发展规划

### 常见问题

**Q: 研究任务找不到？**  
A: 检查是否满足前置条件（图书馆+相关建筑）

**Q: 研究失败了怎么办？**  
A: 当前版本研究不会失败，只是需要时间

**Q: 可以同时进行多个研究吗？**  
A: 可以，只要有足够的NPC和材料

**Q: 研究报告可以交易吗？**  
A: 理论上可以，但非常珍贵，不建议交易

**Q: 高级建筑可以再升级吗？**  
A: 当前版本暂不支持，未来可能添加
