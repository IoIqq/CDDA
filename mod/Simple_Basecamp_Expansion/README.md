**版本**：v0.1.0  
**作者**：IoIqq  
**类别**：建筑/营地扩展  
**兼容性**：CDDA 实验版（2026.6.1+）

---

## 📖 简介

Simple Basecamp Expansion 是一个为 Cataclysm: Dark Days Ahead 设计的营地扩展模组。本模组添加了 4 个新的营地扩展建筑，为玩家提供更多的工作空间和功能。

**核心特点**：
- ✅ **只使用游戏原版物品** - 不添加任何新物品，完全兼容原版
- ✅ **平衡设计** - 材料需求和建造时间经过仔细平衡
- ✅ **模块化设计** - 每个扩展独立，可按需建造
- ✅ **功能完整** - 解锁相应的营地任务和配方

## 🏗️ 扩展建筑

### 1. 🔨 简单铁匠铺（Simple Smithy）

**功能**：提供基础的金属加工能力

**包含设施**：
- 铁砧（f_anvil）
- 熔炉（f_forge）
- 工作台（f_workbench）
- 货架（f_rack）
- 储物柜（f_locker）

**解锁功能**：
- `blacksmith` - 铁匠配方组

**建造要求**：
- 技能：制造 4 级
- 时间：8 小时（480 分钟）
- 工具：2 级锤子、1 级锯子
- 材料：
  - 30 个 2x4 木板
  - 15 个原木
  - 60 个钉子
  - 4 个管道
  - 10 个金属板
  - 20 个石头

**建筑尺寸**：6x6

## 📥 安装方法

### 方法 1：手动安装

1. 下载本模组
2. 将 `Simple_Basecamp_Expansion` 文件夹复制到 CDDA 游戏目录的 `data/mods/` 文件夹中
3. 启动游戏
4. 在创建新世界时，在模组选择界面启用 "Simple Basecamp Expansion"

### 方法 2：从源码安装

```bash
cd <CDDA游戏目录>/data/mods/
git clone <仓库地址> Simple_Basecamp_Expansion
```

## 🎮 使用方法

### 前置条件

在建造这些扩展之前，你需要：
1. 建立一个营地（Basecamp）
2. 将营地升级到至少 **4 级**（faction_base_camp_4）
3. 准备好相应的建造材料

### 建造步骤

1. 在营地菜单中选择 "扩展营地"（Expand Base）
2. 选择要建造的扩展类型
3. 选择建造位置（通常是 field 地形）
4. 确认材料和工具需求
5. 开始建造

### 建造顺序建议

推荐的建造顺序：
1. **仓库** - 提供存储空间，方便管理材料
2. **铁匠铺** - 解锁金属加工，制作工具和武器
3. **木工坊** - 提供木工能力
4. **厨房** - 改善食物管理

## ⚙️ 技术信息

### 文件结构

```
Simple_Basecamp_Expansion/
├── modinfo.json           # 模组信息
├── recipes.json           # 建造配方
├── mapgen.json            # 地图生成
├── recipe_groups.json     # 配方组注册

├── ALLOWED_ITEMS.md       # 物品白名单
```

### 命名规范

所有配方 ID 遵循以下格式：
- 起始配方：`faction_base_<名称>_simple_0`
- 建造配方：`faction_base_<名称>_simple_1`
- 地图生成：`faction_base_<名称>_simple_1_mapgen`
