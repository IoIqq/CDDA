## 🔬 详细测试步骤

### 测试场景1：完整研究流程

#### 步骤
1. **建造图书馆**
   ```
   营地任务 → 建造 → 图书馆
   等待完成
   ```
   **检查**：blueprint_provides包含knowledge_storage

2. **建造铁匠铺**
   ```
   营地任务 → 建造 → 简单铁匠铺
   等待完成
   ```
   **检查**：blueprint_provides包含blacksmith

3. **准备研究材料**
   ```
   收集：
   - 制造学教科书 x1
   - 化学教科书 x1
   - 钢块 x30
   - 铜 x20
   ```

4. **执行冶金研究**
   ```
   营地任务 → 研究 → 研究：高级冶金学
   等待12小时
   ```
   **检查**：
   - 显示"你完成了冶金研究！"消息
   - 获得research_report_metallurgy
   - research_count变量+1

5. **触发里程碑**
   ```
   完成第一项研究后
   ```
   **检查**：
   - 显示"【里程碑】你完成了第一项研究！"
   - 获得first_research_trophy
   - 获得tech_pioneer_blessing效果

6. **建造高级铁匠铺**
   ```
   营地任务 → 建造 → 高级铁匠铺
   消耗research_report_metallurgy
   等待12小时
   ```
   **检查**：
   - 建筑成功建造
   - 提供advanced_blacksmith功能

7. **使用高级配方**
   ```
   制作 → 高效炼钢（高级）
   ```
   **检查**：
   - 产量为3倍（result_mult: 3）
   - 制作时间正确

## 🐛 常见问题排查

### 问题1：研究任务找不到

**可能原因**：
- 未建造图书馆
- 未建造对应基础建筑
- 模组未正确加载

**解决方法**：
1. 检查营地建筑列表
2. 确认blueprint_provides
3. 重启游戏

### 问题2：EoC不触发

**可能原因**：
- 变量未初始化
- 条件不满足
- EoC类型错误

**解决方法**：
1. 检查游戏日志
2. 验证变量状态
3. 确认EoC正确加载

### 问题3：高级建筑无法建造

**可能原因**：
- 缺少研究报告
- 材料不足
- 前置条件不满足

**解决方法**：
1. 确认拥有研究报告
2. 准备充足材料
3. 检查blueprint_requires

### 问题4：消息不显示

**可能原因**：
- u_message语法错误
- 条件未满足
- 变量未设置

**解决方法**：
1. 检查EoC语法
2. 验证条件逻辑
3. 调试变量

## 💡 测试技巧

### 1. 分步测试
- 不要一次测试所有功能
- 按照流程逐步进行
- 记录每一步的结果

### 2. 隔离测试
- 单独测试每个系统
- 排除其他因素干扰
- 便于定位问题

### 3. 重复测试
- 多次重复关键步骤
- 验证结果一致性
- 发现偶发问题

### 4. 边界测试
- 测试极限情况
- 测试错误输入
- 测试异常状态
